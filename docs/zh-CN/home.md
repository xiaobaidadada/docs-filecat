
# 欢迎

这里是 **FileCat** 的官方文档，你可以在这里找到详细的使用说明、应用场景等内容。

**FileCat** 是一个帮助开发者搭建文件服务器的工具，内置了丰富且对程序员非常实用的功能，例如：

- 带权限控制的远程终端（支持 RDP 桌面和终端命令拦截）
- SSH 代理
- 在线文件浏览、编辑、解压缩
- 白板绘图功能
- 自动化构建
- 系统资源监控
- Docker 管理
- 超大日志文件实时查看

当前 FileCat 可运行在 **Linux** 和 **Windows** 系统上，**不支持 macOS 系统**。
